# Implementation Plans

This directory contains TDD-based implementation plans generated by `/plan-tdd`.

Plans break down large features into GitHub issue-sized tasks that follow Test-Driven Development principles.

## Plan Format

Each plan file follows the pattern: `plan-{id}.md`

Plans include:
- Task breakdown with complexity sizing (S/M/L)
- Dependency graph and implementation phases
- Test requirements for each task
- Acceptance criteria
- TDD guidance (Red-Green-Refactor)

## Usage

Generate a plan:
```bash
./adws/adw_plan_tdd.py "Add user authentication with JWT"
```

Plans are designed for execution by Claude Code agents.
